## 사용법 ##
------------------

1. 위 폴더를 안드로이드 롬 소스의 bootable/recovery/gui/theme/extra-languages 폴더에 넣어주세요.

2. 빌드하고자 하는 기기의 BoardConfig.mk에 아래 구문을 추가해주세요:

     TW_EXTRA_LANGUAGES := true

3. 이제 빌드해서 기기에 적용시켜보면, TWRP 설정창에서 한국어 설정이 가능해집니다.
